## Overview

Press `?` at any time to open the keyboard shortcuts reference inside the app. On macOS, `Ctrl` is replaced by `Cmd`.

## General

| Shortcut | Action |
|----------|--------|
| `Ctrl + K` | Open global search |
| `Ctrl + Enter` | Send request |
| `Ctrl + S` | Save request |
| `Ctrl + T` | New tab |
| `Ctrl + W` | Close current tab |
| `?` | Show keyboard shortcuts |

## Tabs

| Action | How |
|--------|-----|
| Rename tab | Double-click the tab title |
| Reorder tabs | Drag within pinned/unpinned groups |
| Tab options | Right-click for pin, close, close others, close all |

## Sidebar

| Action | How |
|--------|-----|
| Rename request | Double-click the request name |
| Reorder | Drag requests and collections |

## Response viewer

| Shortcut | Action |
|----------|--------|
| `Ctrl + F` | Find in response body |
| `Enter` | Next search match |
| `Shift + Enter` | Previous search match |
| `Esc` | Close search |

## Global search

Press `Ctrl + K` to open the command palette. It searches across:

- **Saved requests** — by name, URL, and method
- **History entries** — by URL and method

Use arrow keys to navigate results, `Enter` to open, and `Esc` to close.

When the search field is empty, recently used items are shown.
